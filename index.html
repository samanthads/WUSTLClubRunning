WashU Club Running

Welcome to the website of WashU Club Running! We are a NIRCA running club here at Washington University in St. Louis. We are co-ed and run both local and regional races, but most of all we just love spending time together. Come for a run with us!

Come Run With Us!

Meet us at the clock tower for a run at 5:30pm Sunday-Thursday! All abilities welcome. No commitment necessary. Come run with us if all you want is a running buddy, if you're training for a marathon, and anything in between.
